LakAI est une plateforme web développée dans le cadre d'un projet académique. Son objectif est de faciliter la recherche, la location, l'achat et l'estimation de biens immobiliers dans la ville des Cayes, en Haïti.

Le site propose une interface moderne, intuitive et responsive permettant aux utilisateurs de consulter les différentes propriétés disponibles, de demander une estimation de leur bien et de contacter facilement l'agence.
## Charte graphique

L'interface de LakAI repose sur une palette de trois couleurs principales afin de garantir une identité visuelle cohérente et professionnelle.

- **Vert foncé (#2D5016)** : couleur principale représentant la confiance, la stabilité et le secteur immobilier. Elle est utilisée pour les boutons, les titres, la navigation et certains arrière-plans.
- **Beige (#D4B896)** : couleur secondaire apportant une touche d'élégance et de modernité. Elle est utilisée pour les éléments de mise en valeur, les bordures, les effets de survol et certains arrière-plans.
- **Blanc (#FFFFFF)** : utilisé comme couleur neutre pour améliorer la lisibilité, créer un contraste avec les autres couleurs et mettre en valeur le contenu.



## Utilisation de Bootstrap

Le framework Bootstrap 5 a été utilisé uniquement pour certains composants afin de faciliter le développement tout en conservant une identité graphique personnalisée.

Bootstrap a notamment été utilisé pour :

- les fenêtres modales (Modal) permettant d'afficher les détails des propriétés ;
- les carrousels (Carousel) utilisés pour présenter plusieurs images d'un même bien immobilier ;
- certaines classes utilitaires facilitant l'alignement et l'espacement des éléments.

L'ensemble de la mise en page, des couleurs, des animations, des cartes, de la navigation et du design général a été réalisé principalement en HTML5 et CSS3 afin de conserver une interface unique et adaptée au projet LakAI.

## Technologies utilisées

- HTML5 pour la structure des pages.
- CSS3 pour la mise en forme et les animations.
- Bootstrap 5 pour certains composants interactifs.
- Font Awesome pour les icônes.

## Choix du design

Le design du site privilégie une interface moderne, épurée et intuitive. Les espaces, les contrastes et les animations ont été pensés pour faciliter la navigation des utilisateurs tout en mettant en valeur les propriétés immobilières proposées par l'agence LakAI.
